## Dependencies:
* Java 8+
* environmental variables for database connection:
    * DB_USERNAME
    * DB_PASSWORD
    * DB_URL
    
    for example: 
    * DB_USERNAME=postgres
    * DB_PASSWORD=postgres
    * DB_URL=jdbc:postgresql://localhost/socratica
